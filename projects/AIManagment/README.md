# AIManagment

Минимальный локальный MVP для управления AI-задачами команды.

## Что уже есть

- дашборд с ключевыми метриками;
- форма добавления AI-задачи;
- канбан по статусам;
- фильтрация по статусу, приоритету и текстовому поиску;
- хранение состояния в `localStorage`.

## Структура

- `docs/` — документы по продукту и плану реализации;
- `app/` — статическое приложение (`index.html`, `styles.css`, `app.js`, `state.js`);
- `tests/` — проверки бизнес-логики состояния.

## Как открыть приложение

Достаточно открыть файл `app/index.html` в браузере.

## Проверки

```bash
cd projects/AIManagment
npm test
```
