# AIManagment

Минимальный локальный MVP для управления AI-задачами команды.

## Что уже есть

- дашборд с ключевыми метриками;
- форма добавления AI-задачи;
- канбан по статусам;
- быстрые статус-вкладки для мобильной навигации по доске;
- фильтрация по статусу, приоритету и текстовому поиску;
- хранение состояния в `localStorage` и локальный журнал изменений по задачам;
- документы для перехода к backend-этапу: модель данных, API и интеграции.

## Структура

- `docs/` — документы по продукту, плану реализации и backend-подготовке;
- `app/` — статическое приложение (`index.html`, `styles.css`, `app.js`, `state.js`) с лентой изменений;
- `tests/` — проверки бизнес-логики состояния.

## Документы backend-этапа

- `docs/data-model.md` — целевая серверная модель данных;
- `docs/api-integrations.md` — API v1, сценарии интеграций и порядок внедрения.

## Как открыть приложение

Достаточно открыть файл `app/index.html` в браузере.

## Проверки

```bash
cd projects/AIManagment
npm test
```
