- Дата начала: 2026-03-19

## Цель проекта

Создать понятное приложение для управления AI-задачами, экспериментами и
рабочими статусами внутри команды.

## Контекст

- Нужен простой MVP, который можно открыть локально без backend.
- Сначала важно зафиксировать структуру данных и UX, а не строить сложную архитектуру.
- Основной сценарий — учёт AI-инициатив, моделей, владельцев и прогресса по задачам.

## Ограничения и допущения

- Первая версия работает полностью на клиенте.
- Хранение состояния выполняется через `localStorage`.
- В первой версии не реализуются авторизация и внешние интеграции.
