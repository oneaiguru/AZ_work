
Apartment Blender-import package (OBJ+MTL)

Files included:
- apartment_model.obj
- apartment_model.mtl
- README.txt

Units: meters. Blender import-ready (Wavefront OBJ).

Parameters used (you can edit if needed):
- Outer footprint: 2800.0 mm x 2800.0 mm
- Wall thickness: 120.0 mm
- Room height: 2530.0 mm
- Window: bottom offset 480.0 mm, height 1570.0 mm, width 940.0 mm, start along west wall 470.0 mm
- Door: height 2100 mm, width 900.0 mm (assumed), start along south wall 1900.0 mm

How to open:
1. Open Blender.
2. File → Import → Wavefront (.obj) → apartment_model.obj
3. Objects: Floor, Wall_North, Wall_South, Wall_East, Wall_West, Window_Glass, Door_Panel
Notes: Walls are full boxes; window and door are separate thin panels. To make cutouts, use Boolean modifiers in Blender or ask me to produce booleaned geometry.
