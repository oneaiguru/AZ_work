
Apartment package generated from user's sketch/photo and notes.

Included files:
- apartment_scheme.xml    : Full XML scheme (plan + vertical section) with confidence attributes.
- plan_view.svg           : Top-down plan SVG (outer dims, walls, window and door approximations).
- vertical_section.svg    : Vertical section SVG (floor to ceiling, window and door heights).
- source_image.jpg        : Original photo you uploaded (copied to package).
- README.txt              : This file.

Notes:
- Horizontal plan coordinates (exact X/Y) and some widths were inferred from the sketch and have 'low' or 'medium' confidence.
- Confirm actual measured widths (window horizontal width, door width, wall thicknesses) if you need construction-ready drawings.
- If you provide a high-res PDF/DWG or measured dimensions I will update coordinates and set confidence to high.

How to use:
- Open the SVG files in a browser or vector editor to view/print.
- XML is machine-readable for further processing.
