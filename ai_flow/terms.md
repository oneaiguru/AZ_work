# Термины

- Runbook: пошаговая операционная инструкция для типовых задач/инцидентов.
- Playbook: сценарий с ветвлениями для разных ситуаций.
- SOP (Standard Operating Procedure): формализованная регламентная инструкция.
- Checklist: короткий перечень шагов без подробностей.
- Troubleshooting guide / KB article: справка с типовыми проблемами и решениями.
- Escalation policy / on-call handbook: правила и контакты для эскалации инцидентов.
- Incident response plan: план действий во время инцидента (роли, коммуникации, шаги).
- DRP/BCP (disaster recovery / business continuity): планы восстановления и продолжения работы после крупных сбоев.
